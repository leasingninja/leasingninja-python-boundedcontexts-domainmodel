# leasingninja-python-boundedcontexts-domainmodel

The LeasingNinja in Python with DDD style bounded contexts and domain model.

## Daily Development

```fish
python3 -m venv .venv
source .venv/bin/activate.fish
pip install pytest
pytest
```

## When Python setup messes up

```fish
rm -rf .venv
python3 -m venv .venv
```
